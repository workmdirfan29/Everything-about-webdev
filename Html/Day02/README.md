# List Item Example

## Ordered List

```html
<ol>
    <li>HTML</li>
    <li>CSS</li>
    <li>JS</li>
    <li>React</li>
</ol>
```
## Unordered List
```html 
<ul>
    <li>Peter</li>
    <li>John</li>
    <li>John Doe</li>
    <li>Erik</li>
</ul>
```
### Nested List
```html
<ol>
    <li>
        Peter
        <ul>
            <li>Node.js</li>
            <li>SQL</li>
            <li>Python</li>
            <li>GoLang</li>
        </ul>
    </li>
</ol>
```
